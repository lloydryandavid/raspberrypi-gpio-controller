# raspberrypi-gpio-controller


:+::+: Raspberry PI Pin Wiring :+::+:


RPi - L298N Motor Module

+-------------------------+-------------------------------+
|  Rpi                    |  L298N MOTOR DRIVER MODULE    |
+-------------------------+-------------------------------+
|  #39 (GROUND)           |  GROUND                       |
|  #36 (SOFTWARE PIN_27)  |  INPUT_1                      |
|  #38 (SOFTWARE PIN_28)  |  INPUT_2                      |
+-------------------------+-------------------------------+
